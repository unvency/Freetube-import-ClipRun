# Freetube-Playlist import
Creates [FreeTube](https://freetubeapp.io/) .db playlist files from a list of youtube video urls (only .csv files exported from 'Google takeout').

Just copy the content of the YTplaylist2FreeTubeDB.py and run it locally, or, if you don't have Python installed, use the easy way via cliprun.com or a similar Online-Python-Runner.
